# BattleTypingGame
[ガバタイピングゲーム](https://ne30megane.github.io/SimpleTypingGame/)を作り変えて作成した、タイピングで敵を倒すゲームです。

## 2019/10/28 更新
最低限のゲームの中身を実装。<br>
難易度を選んでゲームをスタートすると、それに応じた数の問題が出題され、タイプミス５回以内に全部タイピングできればクリアとなる。

## 2019/10/30 更新
モンスターのイラストを実装。<br>
ゲームバランスとページレイアウトの調整。
